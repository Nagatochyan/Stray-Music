# Stray-Music
音楽流す最低限の機能が入ってます。<br> 
![brandbird](https://user-images.githubusercontent.com/94958239/225283575-617d626e-e76d-4c55-921b-58d352e4acd7.png)
# 機能
- [x] mp3再生
- [x] 連続再生
- [x] ランダム再生
- [x] 一時停止/再開
- [x] スキップ（バグあるかも）
- [x] 音量調節
- [x] 現在再生中の曲を表示
- [x] mp3の含んだフォルダー読み込み
- [x] 曲一覧を表示
- [x] discord ステータス(richpresence)に表示↓<br>
